# Project Overview
This service allows tracks pairs of managers as a duo and computes a combined points total across FPL gameweeks. It fetches individual player points via the FPL API, aggregates them for partnered players, and exposes a leaderboard UI to view rankings and history.

# Features
- Aggregate and rank total points for each duo
- Responsive React-based UI
- RESTful Express API
- Deployable to AWS Elastic Beanstalk
